# WanderLust-A-Full-Stack-Web-based-Property-rental-Booking-System
Wanderlust is a full-stack web application for property rentals and bookings, designed to provide a seamless experience for both property owners and travelers. The platform enables users to browse, book, and manage properties, while owners can list and manage their properties efficiently.

Wanderlust 🌍🏡

Wanderlust is a full-stack web application for property rentals and bookings, designed to provide a smooth experience for both travelers and property owners. Users can browse and book properties, while owners can manage listings efficiently.

Features ✨

User Authentication: Sign up, log in, and manage profiles securely.

Property Listings: Owners can add, edit, and remove properties with images, descriptions, and pricing.

Search & Filter: Find properties by location, price, and amenities.

Booking System: Real-time booking and availability management.

Reviews & Ratings: Users can leave reviews and ratings for properties.

Responsive Design: Works on mobile and desktop devices.

Tech Stack 🛠️

Frontend: React.js, HTML, CSS, JavaScript

Backend: Node.js, Express.js

Database: MongoDB
